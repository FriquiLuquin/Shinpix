Shinpix

How to play Shinpix

A mysterious pixelart image is hidden in this grid and you have to find it. Activate all the correct tiles to reveal the secret image.

Rules

  • Click a tile to flip it. This will activate or deactivate it
  • Each tile has neighbours: 3 (corner), 5 (border) or 8 (the rest)
  • Each tile indicates how many of those neighbours must be activated
  • Use Minesweeper logic to deduce and activate all the correct tiles

Shinpix

How to play

A mysterious pixelart image is hidden behind this grid and you have to find it.

The rules are very similar to Minesweeper:

  • You can activate a tile by clicking it
  • Some tiles must be activated. They will reveal the hidden image
  • The rest must reamin disabled
  • Each tile has 8 neighbours surrounding it
  • The number inside each tile tells you how many activated neighbours the tile must have
  • Your task is to use these clues to deduce and activate all the correct tiles
